# Danny Diner SQL Case Study

This project showcases SQL queries used to analyze data for Danny's Diner, focusing on customer behavior, spending, and popular menu items.

## SQL Queries Included:
- Total amount spent by each customer
- Number of days visited by each customer
- First item purchased by each customer, etc.

## How to Use
1. Clone the repository.
2. Open the `index.html` file in your browser to view the queries and results.

## Tools Used
- SQL
- HTML/CSS for frontend visualization
